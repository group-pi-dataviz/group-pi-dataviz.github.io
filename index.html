Visualizing Conflict and Human Suffering through Data

Group π - Alessandro Acquilino (5198425), Fabrizio Sardo (5250158), Stefano Passaggio (4875279)

This project aims to visualize data related to conflict and human suffering around the world. The data used in this project is sourced from the Armed Conflict Location & Event Data Project (ACLED), which provides detailed information on political violence and protest events across the globe. The visualizations created in this project help to shed light on the patterns and trends of conflict, enabling a better understanding of its impact on affected populations. Source
This project was developed as part of the Data Visualization course at the University of Genoa by students. Therefore it is important to double-check the data and graphs presented, as they may contain errors or inaccuracies.
